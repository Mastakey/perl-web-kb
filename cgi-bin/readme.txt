This folder contains all scripts and executables.
It may be the case that the cgi-bin directory is shared on a web host, in which case it will be located in the folder web/cgi-bin/skeleton instead of web/skeleton/cgi-bin
Library files/Modules downloaded from the web are located in the lib directory.
The automation directory will contain batch and shell scripts used for automation typically generation of static content [html files].
The web directory will contain .pl files that contains the web content
